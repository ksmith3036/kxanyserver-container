
Transfer the file kxany.docker, as well as the conf directory to the docker host.

Execute the following command
    sudo docker load -i kxany.docker

Start container:
    docker run -d -p 443:443 --name happy_knox happy-knox:1.0

Start / stop:
sudo docker container stop happy_knox
sudo docker container start happy_knox


Check logs
-------------
sudo docker exec -it happy_knox /bin/bash
  cd logs
  ls

Update certificate
-------------------
NB! This is just meant as an example, documenting one way to do it.

Update certificate in conf directory and edit server.xml to change filename and password.
Build a new docker image:
  sudo docker build -t happy-knox:1.1 -f Newcert.Dockerfile .
  sudo docker container stop happy_knox
  sudo docker rm happy_knox
  sudo docker run -d --name happy_knox -p 443:443 -v ~/knox/conf:/usr/local/tomcat/conf happy-knox:1.1
  
As an alternative to updating the docker image, one might mount the files into the image:
  sudo docker run -d --name happy_knox -p 443:443 -v ~/knox/conf:/usr/local/tomcat/conf happy-knox:1.0
